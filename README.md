# Conversor de Monedas - Challenge Alura

Proyecto sencillo hecho en Java para practicar lógica de programación.

## Funcionalidad

El programa permite convertir monedas usando un menú en consola.

Opciones:
- USD a MXN
- MXN a USD
- USD a EUR
- EUR a USD

## Cómo ejecutar

1. Abrir el proyecto en IntelliJ IDEA
2. Ejecutar el archivo Main.java
3. Usar el menú en consola
